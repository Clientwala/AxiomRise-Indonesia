# PT AxiomRise Teknologi Indonesia

This repository contains the source code for the PT AxiomRise Teknologi Indonesia corporate website.

## Project Structure

- `frontend/`: Contains the React/Vite web application for the corporate landing page.
- `backend/`: Placeholder directory for future backend API services.

## Technologies Used (Frontend)

- **Framework**: React via Vite
- **Styling**: Vanilla CSS with modern Glassmorphism aesthetics
- **Icons**: Lucide React
- **Theme**: Dark cyber-tech with glowing accents

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm

### Running the Frontend locally

1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and visit `http://localhost:5173`.

## Deployment

The frontend is ready to be deployed to Vercel or any other static hosting platform. 
**Note for Vercel:** Ensure you set the **Root Directory** to `frontend` in your Vercel project settings to avoid 404 errors during deployment.
